Automatic cat feeder

Developed an automatic cat feeder using Arduino, which includes features such as taking a photo when the cat eats and sending it to the phone, as well as scheduling the feeding times for automatic on/off operation.

Equipment used: arduino uno r3 board, servo motor, lcd display for arduino, clock signal module, esp 32 cam board, sensor hc-sr04, step down module

Working section: It is divided into 2 parts. The first part is the food supply section. It uses Arduino Uno R3 to control the servo, LCD, and clock signal. The second part, which is the part for connecting to WiFi, uses ESP32-CAM.
